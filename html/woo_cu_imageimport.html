Import images from URLS

This feature downloads images from URLS found in your product metadata

Enter the names of the metafields you wish to search, separated by commas
The first field that contains a url for each product will be set as the main product image, any others will be set as product gallery images
Click 'Count' to check how many products have images to import. Click 'Reset' to restore columns that have been previously imported

Please note that this process will overwrite existing images where data is found!

Click 'Process' to start the import process.
