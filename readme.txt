== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/wc_cu_plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the ChannelUnity tab to sign up for a new account. 
4. If you have an account already visit https://kb.channelunity.com/article/connect-woocommerce/ for instructions to link your account.

Please note: This plugin communicates with the remote service at my.channelunity.com.
It also validates email addresses via www.stopforumspam.com
